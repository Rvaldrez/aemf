# aemf

Aplicação web desenvolvida em PHP para gerenciamento e integração de dados.

## Requisitos

- PHP 7.4+
- MySQL 5.7+
- Composer

## Configuração

1. Clone o repositório
2. Configure o banco de dados em `config.php`
3. Instale as dependências com `composer install`

## Estrutura

```
aemf/
├── api/           # Endpoints da API
├── config.php     # Configurações da aplicação
├── conexao_bd.php # Conexão com o banco de dados
└── index.php      # Ponto de entrada da aplicação
```
